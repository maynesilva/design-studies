# 🎨 Design Studies

Este repositório é dedicado aos meus estudos de **HTML + CSS** aplicados a diferentes estilos de design.  
A cada semana escolho um estilo e crio um projeto prático para explorar suas características visuais.

---

## Plano de Estudos
- **Semana 1:** Neo Brutalism  
---

## Estrutura
Cada estilo tem sua própria pasta com:
- `index.html` → página principal
- `style.css` → estilos aplicados
- `README.md` → explicação do estilo e minhas observações

## Objetivo
- Praticar diferentes estilos de design.   
- Documentar aprendizados e boas práticas de HTML + CSS.
- Usar, também, as boas práticas de UX/UI Design com cada estilo.
