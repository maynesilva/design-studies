## O que eu apliquei neste projeto
- Fundo com **padrão de bolinhas** usando `radial-gradient`.  
- Cards com **bordas grossas e sombras duras**.  
- Círculos coloridos para destacar palavras-chave (*being intentionally unpolished, LOUD, Confident*).  
- Botão centralizado com cores fortes e hover.  
- Tipografia **Lexend Mega**, pesada e chamativa.
